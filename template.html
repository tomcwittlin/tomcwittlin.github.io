Example wrapper column. Did you know the class 'columns' default to 'small-12' so you don't need to include both. Also instead o using buffer columns it's more beneficial to use 'push' and 'pull' classes to move 'columns'

This is an example of a nested row. Make sure the stack goes row-columns-row-columns-row-columns otherwise the grid will all fall out of position and is a pain to fix.

My advice with the grid is to put the columns class first, then the column numbers in order of small, medium, then large, then add any other classes afterwards. This makes it easier to read. I also advise keeping row and column divs separate from any other div where possible (this isn't optimal though).
